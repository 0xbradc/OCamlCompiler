Developed by Brad Campbell in the Spring of 2021.



# Getting Started
1. [Install Homebrew](http://brew.sh).

2. Run the following for needed dependencies.
  ```
  brew update
  brew install xquartz
  brew install pkg-config
  brew install gpatch
  brew install opam
  ```
